# Build Instructions

1. 프로젝트 디렉토리로 이동합니다. 
    예: cd C:\Users\LG-PC\Desktop\Git_Clone\RVC_Control_C

2. 아래 명령어를 실행하여 소스 코드를 컴파일합니다:
    gcc -o rvc_control.exe main.c

3. 프로그램을 실행합니다:
    rvc_control.exe
